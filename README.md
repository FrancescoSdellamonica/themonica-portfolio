# theMonica — Portfolio (GitHub Pages)

Questo repository contiene il portfolio fotografico di **Francesco Saverio Della Monica**.

## Pubblicazione
Il sito è ospitato su **GitHub Pages** e accessibile all'indirizzo:
`https://<tuo-username>.github.io/themonica-portfolio/`

Per pubblicare:
1. Vai su **Settings → Pages**.
2. Imposta `Deploy from a branch → main → /(root)`.

## Modifiche
- Le immagini si trovano in `assets/images/`.
- Il layout è definito in `index.html`.
- Puoi cambiare testi e foto liberamente, oppure aggiungere link ai tuoi NFT su OpenSea.
